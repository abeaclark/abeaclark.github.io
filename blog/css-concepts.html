CSS display attribute

The basics

Elements in HTML are all rectangles of different sizes. The CSS display attribute governs how these rectangles behave in relation to eachother. The display attribute comes with several options, below are the most commonly used:

  • Elements sit on the same line and do not interrupt the flow of the document
  • This would be used on attributes like <strong> and <em> which are designed to appear within a line of text, but not force it to insert any additional vertical space
  • You can use margin/padding attributes to insert horizontal space
  • Elements sit on the same line, but are treated as individual rectangles
  • Margin/padding attributes would be evaluated for all 4 sides of an object
  • Elements are forced onto a new line
  • If not width is specified, block elements default to 100%
  • Elements are "hidden" from the screen and will not appear to the viewer or effect subsequent formatting
