Scope

Fan site for OKOREN -- Park City, Utah based music producer

Key features

  • Users can listen to the latest tracks pulled from the Soundcloud API
  • Fan sign-up form backed by Google Docs for ease of use for non-technical administrators
  • Seamless embedded Youtube video with vanity cover

Technologies used

  • Ruby on rails -- Currently a 1-page app, but needed ability to scale
  • Bootstrap CSS/Javascript library
